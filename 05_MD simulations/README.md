# Description

This folder contains the Molecular Dynamics Simulation outputs for all 8 TPO-ligand complexes performed using Schrödinger 
Desmond (NPT ensemble, 300K, 10ns simulation length).

Subfolders:
- RMSD_plots/ — Root Mean Square Deviation trajectories showing 
  protein and ligand stability over the 10ns simulation for all 
  8 complexes
- RMSF_plots/ — Root Mean Square Fluctuation plots showing 
  per-residue flexibility of TPO on interaction with each ligand
- protein_ligand_contacts/ — Interaction fingerprint plots 
  showing hydrogen bonds, hydrophobic, ionic and water bridge 
  contacts between TPO and each ligand throughout the simulation

Simulation Parameters:
- Software: Schrödinger Desmond
- Ensemble: NPT
- Temperature: 300K
- Simulation length: 10ns
